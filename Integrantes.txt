Instrucciones para Clonar y Levantar el Proyecto

1. Requisitos Previos
Antes de clonar el proyecto, asegúrate de tener instalados los siguientes programas en tu sistema:

Node.js (que incluye npm):
node js en la vercion 22.12
Clonnar el proyecto:
    1)git clone url del repositorio
    2)npm install
    3)npm run dev
npm install -g json-server
